# Aila4Cuts Photobooth Web App

Professional browser-based photobooth system designed for events.
Built for simple event setup, clean photo strip output, and smooth customer interaction.

This project powers the **Aila4Cuts Photobooth experience** used in live events.

---

# Overview

This web application allows an event operator to:

1. Setup an event name
2. Upload a custom photo strip border layout
3. Start the photobooth session
4. Allow customers to capture photos
5. Customize and download their photo strip
6. Reset the booth for a new event

Everything runs directly in the browser using the device camera.

---
